# ALFRED SECURE WEB SYSTEM & API

ALFRED now features a professional web interface with a secure user dashboard.

## 1. Web Access (Console)
Open your browser and navigate to:
http://127.0.0.1:5000

- **Homepage**: View project functions, workflow, and technical architecture.
- **Signup**: Create an account to generate your unique API Key.
- **Login/Dashboard**: Access your personal console to view your key and toggle API access ON/OFF.

## 2. API Endpoints

### [POST] /execute
Execute system commands (Requires: 'admin', 'coder' OR a user with 'Access Enabled' in dashboard).

Payload (JSON):
{
    "username": "your_username",
    "password": "your_password",
    "api_key": "YOUR_API_KEY",
    "command": "what is the time"
}

## 3. Security & Access Control
- **Authorized Roles**: 'admin' and 'coder' accounts override the manual toggle and always have access.
- **Custom Users**: Must register, login, and ensure "API Key Access" is turned ON in the dashboard to use the API.
